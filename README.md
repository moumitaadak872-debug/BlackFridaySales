# 🛍️ Black Friday Sales Analysis

## 📌 Project Description
The Black Friday Sales Analysis project is a data analytics project that explores customer purchasing behavior during Black Friday shopping events. Using Python and various data analysis libraries, the project performs data cleaning, exploratory data analysis (EDA), and visualization to uncover meaningful patterns in customer demographics, product preferences, and purchase amounts.

The insights generated from this analysis can help businesses understand customer behavior and make data-driven decisions for marketing, inventory management, and sales strategies.

---

# 🎯 Objectives

- Analyze Black Friday sales data.
- Understand customer purchasing behavior.
- Explore the impact of age, gender, occupation, and marital status on purchasing.
- Identify the most popular product categories.
- Discover spending patterns across different customer groups.
- Generate visual insights for better business decision-making.

# 📊 Dataset Information

The dataset contains historical Black Friday sales records collected from a retail store.

Dataset includes information such as:

- User ID
- Product ID
- Gender
- Age
- Occupation
- City Category
- Stay in Current City Years
- Marital Status
- Product Category 1
- Product Category 2
- Product Category 3
- Purchase Amount

These attributes are used to analyze customer demographics and purchasing behavior.

## ⚙ Installation
1. Clone the Repository

```bash
git clone https://github.com/your-username/BlackFriday-Sales-Analysis.git
```

2. Navigate to the Project Folder

```bash
cd BlackFriday-Sales-Analysis
```
3. Install Required Libraries

```bash
pip install pandas numpy matplotlib seaborn
```

or

```bash
pip install -r requirements.txt
```

4. Launch Jupyter Notebook

```bash
jupyter notebook
```

5. Open

```
BlackFriday.ipynb
```

Run all notebook cells sequentially.
